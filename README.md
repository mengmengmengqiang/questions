## <p id="1.3">Lisp</p>
### <p id="1.3.1">问题</p>

使用Lisp语言定义一个函数，用来表示数列T(n)=7T(n/2)+n^2(n>1);T(1)=1,写出函数并使用Lisp软件计算出**n**取值**1024**时函数的值，
要求当输入变量值为负数时返回提示信息(ERROR)。


### <p id="1.3.2">答案以及评分细则</p>

    (defun T (n)
                (cond
                    ((= n 1) 1)
                    ((< n 1) `(ERROR!))
                    (t (+ (* 7 (T (/ n 2))) (* n n)))
                )
    )

    (T 1024)
    返回值为657710813
    (T -1)
    返回值为(ERROR!)
